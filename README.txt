Project - Sentiment Analysis on Harry potter 7th movie:  Deathly Hallows script


Dataset- script.csv : The CSV file contains 4 columns - 
1)Character A - Speaker
2)Character B - Listener
3)Conversation - Dialogue
4)Line	- Line number

Description:

This was my frist porject in R, the main objective of this project was the introduction to Data Mining and R.

Research Question: Why this movie performed poorly as compared to the other movies of the same series？

Hypothesis: The movie has a dark/gloomy/negative plot, therefore, the audience did not like this movie as much.


To prove out Hypothesis right, we did a sentiment analysis of the script and compared it with box-office collection.
